# Canvas Chronicle - Portfolio of Kirubalan V

A high-performance, cinematic portfolio built with React, Three.js, and Framer Motion. This project showcases the journey and technical expertise of Kirubalan V, a Full-Stack Developer and Tool Builder.

## 🚀 Features

- **Cinematic Experience**: Integration of Three.js and Spline for a visual-first approach.
- **Interactive Universe**: A dynamic skills ecosystem with orbital animations.
- **Responsive Design**: Fully optimized for all screen sizes.
- **Advanced Animations**: Powered by Framer Motion and GSAP for smooth transitions.

## 🛠️ Tech Stack

- **Framework**: React 18
- **Styling**: Tailwind CSS
- **Components**: Radix UI / shadcn/ui
- **Animation**: Framer Motion, GSAP
- **3D**: Three.js, @react-three/fiber, @react-three/drei
- **Build Tool**: Vite

## 📦 Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
